# 🚀 Java Web Application Build & Deployment using Maven on EC2

## 📌 Project Overview
This project demonstrates the end-to-end process of building and deploying a Java-based web application using Maven and hosting it on a cloud server. The application is exposed to the internet via a public IP and port, making it accessible through a web browser.

---

## 🧠 Objective
The main objective of this project is to understand how applications are:
- Built using a build tool
- Packaged into deployable artifacts
- Deployed on a cloud-based server
- Exposed to users over the internet

---

## 🛠️ Technologies Used
- Java (Application Development)
- Maven (Build & Packaging)
- AWS EC2 (Cloud Hosting)
- Linux (Server Environment)

---

## ⚙️ Key Concepts Covered
- Build Lifecycle Management using Maven
- JAR Packaging and Execution
- Client-Server Architecture
- Port Exposure and Networking Basics
- Running Applications in Background Processes

---

## 🌐 Application Workflow
1. Source code is written in Java
2. Maven is used to compile and package the application into a JAR file
3. The JAR file is deployed on an EC2 instance
4. The application runs on a specific port (8080)
5. Users access the application via browser using public IP and port

---

## 🎯 Learning Outcomes
- Understanding how build tools automate compilation and packaging
- Gaining hands-on experience with cloud deployment
- Learning how web applications are exposed over a network
- Understanding the basics of application lifecycle in a DevOps environment
